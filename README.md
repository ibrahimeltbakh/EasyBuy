# Easy Buy – E‑Commerce Website

[Live Demo](https://eassybuy.netlify.app/) • [GitHub Repository](https://github.com/ibrahimeltbakh/EasyBuy)

Easy Buy is a modern e‑commerce platform for selling various products, offering responsive design, product browsing, wishlist, and checkout features for users.

## Features

### User Features
- **Home Page**: Engaging UI with featured products.
- **Products**: Browse, filter, and sort products.
- **Wishlist**: Save favorite products for later.
- **Categories & Brands**: Browse products by category or brand.
- **Cart & Checkout**: Add products to cart and complete purchase.
- **Responsive Design**: Works on mobile and desktop.


## Tech Stack
- **Frontend**: ReactJS, Redux, Material‑UI, React Router, React Slick  
- **State & Data Handling**: Redux Toolkit, React Query, Axios  
- **Forms & Validation**: Formik, Yup  
- **Notifications & Alerts**: React Hot Toast, SweetAlert2  
- **UI & Animations**: Emotion, React Spinners, Slick Carousel  
- **Build Tools**: Vite, ESLint  
- **Other**: Responsive Design, Dark/Light Theme support (if implemented)
